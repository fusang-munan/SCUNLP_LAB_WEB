## Project Highlights

- Implemented intent recognition using BERT+BiLSTM hybrid model
- Dialogue state tracking based on attention mechanism
- Q&A generation integrating domain knowledge graphs
- Support for both speech and text multimodal interaction

## Technical Metrics

| Metric         | Performance |
|----------------|-------------|
| Intent Recognition Accuracy | 85.3%  |
| Q&A Matching Accuracy | 92.1%  |
| Response Time      | <1.5s  |
| Concurrent Processing Capacity  | 1000+  |

## Application Results

The results of this project have been deployed and applied in the actual business scenarios of the partner bank, handling over 5,000 customer inquiries daily, improving customer satisfaction by 40%, and reducing the workload of human customer service by 60%.

For more technical details, welcome to download our [Technical Report](/projects/Intelligent-Customer-Service-System-Technical-Report.pdf) or contact the laboratory to discuss cooperation opportunities.
